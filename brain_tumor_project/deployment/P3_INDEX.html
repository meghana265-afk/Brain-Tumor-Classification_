Execution Order

Follow this sequence during the demo (see details):

# Phase 1: Baseline
python brain_tumor_project\src\train_model.py
python brain_tumor_project\src\evaluate.py

# Phase 2: Enhanced (P2)
python brain_tumor_project\src\train_model_enhanced.py
python brain_tumor_project\src\evaluate_enhanced.py
python brain_tumor_project\src\compare_models.py

# Phase 3: Demo
python brain_tumor_project\src\predict.py Testing\glioma\Te-glTr_0000.jpg --both

Data Pipeline

Ingestion → Preprocess → Augment → Train → Evaluate → Predict → Reports

  • Ingestion: `Training/` and `Testing/` folders
  • Preprocess: preprocess.py
  • Augment: within train_model*.py
  • Train: train_model.py, train_model_enhanced.py
  • Evaluate: evaluate.py, evaluate_enhanced.py
  • Predict: predict.py
  • Reports: text files + plots

Environment & Troubleshooting

  • Python 3.10.11, TensorFlow 2.10.0
  • Install deps: pip install tensorflow numpy opencv-python matplotlib scikit-learn seaborn pandas
  • Windows console uses ASCII: we replaced emojis and Unicode bars.
